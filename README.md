ExpenseFlow is a full-stack personal finance management application that enables users to securely manage their income and expenses, categorize transactions, monitor spending habits, and visualize financial summaries through an intuitive dashboard.


A User will have the options to:
// AUTHENTICATION
Register
Login
Logout
// DASHBOARD

View total income
View total expenses
View current balance

// INCOME
Add income
Edit income
Delete income
View income history

// EXPENSES
Add expense
Edit expense
Delete expense
View expense history

// REPORTS
Monthly summary
Category-wise expenses

// PROFILE
Update profile
Change password
